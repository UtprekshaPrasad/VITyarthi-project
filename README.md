##Project Title
Library Management System (LMS)
---

## Overview of the Project
The Library Management System is a Python-based application designed to simplify and automate daily library operations.  
It allows librarians, students, and administrators to manage books, users, issues, and returns efficiently.  
The system uses SQLite as a backend database and provides a simple command-line interface (CLI).

---

## Features
- Add, update, and search books  
- Register and manage users  
- Issue books with due dates  
- Return books with automatic fine calculation  
- Track available vs issued copies  
- View all issue history  
- Prevent issuing when stock is unavailable  

---

## Technologies / Tools Used
- **Python 3**
- **SQLite3** (built-in database)
- **datetime module** for date handling
- **Command Line Interface (CLI)** for user interaction

---

## How to Run
- Ensure Python is installed on your system.
- Download this repository.
- Open a terminal/command prompt in the project folder.
- Run the following command:
   python main.py
